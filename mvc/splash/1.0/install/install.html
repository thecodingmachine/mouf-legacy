Splash install

SVN checkout

Recover the Splash sources to http://www.thecodingmachine.com/svn/splash/trunk and place them in a splah/ directory in your project. For example create your project directory project/ and create the splash directory project/splash/.

Copy install directory

Go to install/ directory (project/splash/install/) and copy the content to the root of your project (project/).

Splash configuration

Copy the config.template.php in the root project directory (project/config.php). Change the config file:

Change the .htaccess file: Options FollowSymLinks RewriteEngine on RewriteBase /project RewriteCond %{REQUEST_FILENAME} !-f # RewriteRule ^(.+)$ index.php/$1 RewriteRule !\.(js|ico|gif|jpg|png|css)$ splash/index.php
