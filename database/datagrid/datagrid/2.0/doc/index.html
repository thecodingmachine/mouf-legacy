Displaying datagrids

The Mouf framework is only an IOC framework. As such, it does not provide any means for displaying datagrids. Hopefully, the Mouf team provides also a range of packages to display datagrids. A Datagrid can be used to display advanced tables in HTML. The grid can usually be sorted, can provide Ajax loading and row expanders...

The datagrid architecture

All grids provided by the Mouf framework are split in 2 parts:

  • The datagrid: it is the object responsible for displaying the grid in HTML
  • The datasource: it is the object responsible for querying data (in database or anywhere else)

A datasource contains

Any kind of datasource can be combined with any kind of datagrid.

The datasource

The Mouf team provides 2 datasources, and of course, you are free to implement your own.

  • The SQL datasource: The SQL datasource is performing a request on the database to retrieve data. It requires the TDBM framework to run
  • The DBM datasource: The DBM datasource is performing a request on a specific table. Very useful if all the data you want to display is part of only one table. It requires the TDBM framework to run

Each datasource implements the XajaDataSourceInterface interface. When implementing your own datasource, you should extend this interface too.

The datagrids

The Mouf team provides one implementation of a datagrid: the jqGrid datagrid, based on the jqGrid jQuery widget.
