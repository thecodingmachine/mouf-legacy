Installing Druplash

Druplash is a component enabling the integration of Mouf and Splash MVC framework with Drupal.

In order to set-up your development environment, you will need to install first Mouf, then Drupal, then enable the required modules.

Druplash v1.0 integrates with Drupal 6.

Druplash v7.x-1.x integrates with Drupal 7.

Step 1: installing Mouf

Start by performing a normal Mouf install. The installation steps are described in the Mouf install guide (TOTO check this link).

Step 2: installing Drupal

In the directory that contains Mouf, install Drupal.

Step 3: download and enable Druplash package in Mouf

Open the Mouf interface (http://[your_server]/[app_directory]/mouf)

Click on Download packages

Enable the Druplash package matching your Drupal version.

Step 4: Install the Drupal modules

In the previous step, we installed the Mouf packages. Now, we must install the Drupal modules. Druplash has a Drupal module. Druplash module can be found in plugins/integration/drupal/druplash/[your druplah version]/drupal_module

Both modules should be copied in the Drupal directory /sites/all/modules.

Finally, you need to enable those modules in Drupal. Log-in as administrator in Drupal, go to the module administration page and enable Druplash.
